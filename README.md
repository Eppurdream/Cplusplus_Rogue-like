# Cplusplus_Rogue-like

- 2학년 1학기에 개발한 프로젝트

```
C++ 언어로 로그라이크 게임의 기능을 하기 위해서 시작하였다

다만

개발기간이 약 3주라는 기간과
개발을 배운지 1년도 안되는 적은 경험으로 인해

스테이지 한개만 제작하게 되었다.
```

---

프로젝트 내부 기능

---

- 전체적인 틀은 처음 시작하는 곳에서 목적지까지 도달만 하면 되는 구성을 가짐

- 맵은 실행할때마다 랜덤으로 생성되며 진행도중 적들이 출몰함

- 적들에게 부딪힐때마다 플레이어의 체력이 줄어드는데 0이 될 경우 게임이 종료된다
